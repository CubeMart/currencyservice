# CubeMart Currency Service

![CubeMart](https://img.shields.io/badge/CubeMart-Currency-0f766e)
![Node.js](https://img.shields.io/badge/Node.js-Service-339933)
![gRPC](https://img.shields.io/badge/gRPC-Currency-1f2937)

Currency conversion microservice that enables multi-currency pricing across CubeMart.

## Responsibilities

- Convert prices between supported currencies
- Return supported currency codes
- Serve pricing conversions over gRPC
- Support frontend and checkout pricing workflows
